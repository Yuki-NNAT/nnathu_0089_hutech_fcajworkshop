### Week 1 Achievements:

* Became familiar with the First Cloud AI Journey program, mentors, and other participants.

* Understood the internship rules, progress-reporting process, and overall learning roadmap.

* Gained a basic understanding of cloud computing and its key benefits, including:

  * Flexible resource scaling.
  * Faster system deployment.
  * Lower initial infrastructure costs.
  * Pay-as-you-go pricing.
  * The ability to operate systems across different regions.

* Learned the differences between the main cloud deployment models:

  * **Public Cloud:** Cloud resources provided and managed by a third-party service provider.
  * **Private Cloud:** Cloud infrastructure dedicated to a single organization.
  * **Hybrid Cloud:** A combination of public and private cloud environments.

* Understood the main cloud service models:

  * **Infrastructure as a Service (IaaS):** Provides infrastructure such as virtual machines, storage, and networking.
  * **Platform as a Service (PaaS):** Provides a platform for developing and deploying applications.
  * **Software as a Service (SaaS):** Provides complete software applications over the Internet.

* Learned the purpose of several main AWS service categories:

  * **Compute:** Provides processing resources and virtual servers.
  * **Storage:** Stores files, application data, and backups.
  * **Database:** Provides managed database services in the cloud.
  * **Networking:** Connects AWS resources and supports network configuration.
  * **Security:** Manages identities, access permissions, and resource protection.
  * **AI/ML:** Supports the development of artificial intelligence and machine learning applications.

* Successfully created an AWS account and accessed the AWS Management Console.

* Learned how to search for AWS services, select a Region, and check basic account information.

* Completed the required program tasks and received **USD 200 in AWS Credit** for learning and project development.

* Understood the importance of monitoring costs and managing resources carefully when using AWS.

* Completed the first week’s objectives and built a foundation for learning about account security, cost management, and hands-on AWS services in the following week.
